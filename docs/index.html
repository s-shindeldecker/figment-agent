E100 Experimentation Targeting — How It Works

Figment is the automated agent that rebuilds and maintains the E100 experimentation target list — the 100 highest-priority accounts for the Experimentation Solutions team. It replaces a manual, stale spreadsheet with a live, multi-source scoring system that refreshes weekly without a human owner.

Design principles
Deterministic
Same inputs + same config = same ranked output. No LLM in the scorer. Weights live in settings.yaml.
Config-driven queries
Tier 2 Cypher queries stored as LD feature flags. Tune competitive targeting without touching code or redeploying.
Two output views
Per-tier ranked tabs in Sheets for focused review. One merged master list for cross-tier prioritization and Slack digest.
Human override layer preserved — AEs, CSMs, and Exp Solutions can flag any account for inclusion, exclusion, or escalation.
